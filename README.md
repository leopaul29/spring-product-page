# product-page

> In this project, I made a webapp to make a CRUD of product with mongoDB

In this webapp we can:
- [x] show a list of product
- [ ] show a single product
- [ ] create a new product
- [ ] edit a product
- [ ] delete a product
- [ ] 404 product not found
